# player-hater
A silly discord bot for the uWaterloo Hockey Club

## Development Access

Only club executives are authorized to develop and make changes to this bot. Further details in the club discord.

## Development Workflow

- **Feature branches & dev branch**: All changes should be committed to the `dev` branch.
- **Main branch updates**: Updates to the `main` branch must be done sparingly and require approval from at least one other developer before merging.
- **Pull Requests**: Use pull requests for code review and approval before merging to `main`.

## Deployment

**Hosting**: The bot is hosted on [justrunmy.app](https://justrunmy.app).

**Automatic Deployment**: A GitHub Action automatically deploys the bot whenever changes are pushed to the `main` branch.

**Deployment Limits**: The bot can only be deployed **3 times in a 24-hour period** due to hosting provider constraints. Plan deployments accordingly.
